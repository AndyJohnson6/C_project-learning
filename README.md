# 🚀 My C Language Learning Journey
 Welcome to my GitHub repository! This is where I document my journey of learning the C programming language, starting from the absolute basics and working my way up to more complex projects.
 ## 📌 About This Repository
 This is a personal training space for me as a beginner. The projects here are simple exercises designed to build a solid foundation in programming logic, problem-solving, and core C concepts. I will be adding new projects and improvements as I continue to learn.
 ## 📂 Project Structure
 All projects are organized by difficulty level for easy navigation:
 ### 🔰 Level 1: Basic Programs
 *Simple input/output, variables, and basic arithmetic.*
 - `01_addition.c`: A basic two-number addition calculator
 - `02_subtraction.c`: A simple subtraction calculator
 - `03_score_checker.c`: A program to determine grades based on scores
 ### ➕ More Projects Coming Soon!
 As I progress, I will add new exercises involving loops, conditionals, arrays, functions, and more. Stay tuned!
